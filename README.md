# SceneCraft

AI-powered content scheduler for Instagram video ads.

## Project Structure

```
scenecraft-app/
├── src/
│   ├── App.jsx        ← Main app (scene-publisher code)
│   └── main.jsx       ← Entry point
├── index.html         ← HTML shell with mobile meta tags
├── favicon.svg        ← App icon
├── package.json       ← Dependencies
├── vite.config.js     ← Build config
└── .gitignore
```

## Setup

```bash
npm install
npm run dev
```

## Deploy
Connected to Vercel — pushes to main branch auto-deploy.
